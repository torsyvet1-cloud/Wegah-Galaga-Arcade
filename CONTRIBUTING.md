# Galaga Arcade - Documentação de Contribuição

Obrigado por seu interesse em contribuir com **Galaga Arcade**! 🎮

## Como Contribuir

### 1. Reportar Bugs

Antes de criar um relatório de bug, verifique a lista de issues - talvez o problema já tenha sido reportado. Se você encontrar um novo bug:

- **Use um título descritivo** para o issue
- **Descreva o comportamento exato**
- **Descreva o comportamento esperado**
- **Inclua screenshots/GIFs** se possível
- **Inclua seu ambiente** (navegador, SO, dispositivo)

### 2. Sugerir Melhorias

- **Use um título descritivo**
- **Forneça uma descrição detalhada** da melhoria sugerida
- **Liste alguns exemplos** de como essa melhoria seria usada

### 3. Pull Requests

PRs são bem-vindas! Por favor:

1. Fork o repositório
2. Crie uma branch com nome descritivo (`git checkout -b feature/nova-feature`)
3. Faça commit das suas mudanças (`git commit -m 'Adiciona nova feature'`)
4. Push para a branch (`git push origin feature/nova-feature`)
5. Abra um Pull Request

### Padrões de Código

- Use JavaScript ES6+
- Adicione comentários significativos
- Mantenha o estilo de código consistente
- Teste em múltiplos navegadores
- Teste em móvel e desktop
